Recipe Management Website

Technologies: ASP.NET and Transact-SQL Database

Project Executive Summary

Gordon Stevens and Erik Taylor (our team) created a recipe website for multiple users, administrators and their recipes.

The interface presents the user with a fresh HTML5/CSS3 look, using the Bootstrap library. A website guest has the option to register for the website or login.

The user has the ability to list all recipes they have access to view, add their own recipes which can be shared later or kept private, they can also delete their own recipes.

An administrator can turn a user into an administrator, add/delete/update user or recipe information, in addition to their standard user abilities.

Project Feature Highlights

Coded using Microsoft Visual Studio 2015

ASP.NET/C# programming

Transact-SQL code developed using Microsoft SQL Server Management Studio (SSMS)

Embedded database as a Master Data File (MDF)

Microsoft ASP.NET
(Excerpt from Wikipedia)

ASP.NET is an open-source server-side web application framework designed for web development to produce dynamic web pages.

It was developed by Microsoft to allow programmers to build dynamic web sites, web applications and web services.

Microsoft Transact-SQL (T-SQL)
(Excerpt from Wikipedia)

Transact-SQL (T-SQL) is Microsoft's and Sybase's proprietary extension to the SQL (Structured Query Language) used to interact with relational databases.

Transact-SQL is central to using Microsoft SQL Server. All applications that communicate with an instance of SQL Server do so by sending Transact-SQL statements to the server, regardless of the user interface of the application.

Active Server Pages .NET FRAMEWORK (ASP.NET)

Microsoft ASP.NET is the main technology we used to create the website.

The user sees the normal webpage and if they click "View Source" in their web browser the user can see the usual HTML and CSS code.

What the user cannot see is the ASP.NET code which stays on the webserver, and is used to make computations, check permissions and delegate what HTML/CSS code should be sent to the user.

This allows the creation of a website that can change depending on what the user inputs, such as they can see their own recipes only after they have logged in.

Microsoft Transact-SQL (T-SQL)

The most common and standardized language for "relational-type" databases is called Structured Query Language (SQL).

The companies Sybase and Microsoft have taken standard SQL and created an elegant version of SQL called "Transact-SQL" which is visually beautiful, and extends a first-class feature set to make it more usable.

The language that the database code is written in is called "Transact Structured Query Language".

The server which runs the T-SQL code is called "Microsoft SQL Server (SQL Server)". A database server is more commonly refered to as a "Relational Database Management System (RDBMS)".

The software a programmer uses to write T-SQL code is called "Microsoft SQL Server Management Studio (SSMS)".
